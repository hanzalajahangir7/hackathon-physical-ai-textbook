# Assessments

## Grading Structure

| Component | Weight | Description |
|-----------|--------|-------------|
| Weekly Labs | 30% | Hands-on ROS 2 and simulation exercises |
| Midterm Project | 20% | ROS 2 + Gazebo integration project |
| Final Capstone | 35% | Autonomous humanoid with VLA |
| Participation & Quizzes | 15% | Class engagement and weekly quizzes |

## Weekly Labs (30%)

- **Week 3**: First ROS 2 node
- **Week 4**: Publisher-Subscriber system
- **Week 5**: Service and action servers
- **Week 6**: Gazebo simulation setup
- **Week 7**: URDF robot model
- **Week 8**: Isaac Sim introduction
- **Week 9**: VSLAM implementation
- **Week 10**: Nav2 path planning
- **Week 11**: Humanoid locomotion
- **Week 12**: Manipulation tasks

## Midterm Project (20%)

**Due**: End of Week 7

Create a mobile robot in Gazebo that:
- Uses ROS 2 for control
- Has LiDAR and camera sensors
- Navigates a custom environment
- Avoids obstacles autonomously

## Final Capstone (35%)

**Due**: End of Week 13

Build an autonomous humanoid that:
1. Accepts voice commands
2. Plans optimal paths
3. Navigates dynamic environments
4. Identifies objects with computer vision
5. Manipulates objects with robotic arms

## Participation & Quizzes (15%)

- Weekly quizzes on lecture material
- Active participation in discussions
- Helping peers in forums
