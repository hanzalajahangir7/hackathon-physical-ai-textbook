# Cloud Lab Setup

Use cloud resources for robotics development.

## AWS RoboMaker

- Managed ROS 2 environment
- Scalable simulation
- Pay-per-use pricing

## Google Cloud

- Compute Engine for simulations
- TPU/GPU access
- Kubernetes for deployment

## NVIDIA Omniverse Cloud

- Isaac Sim in browser
- No local GPU needed
- Collaborative development
