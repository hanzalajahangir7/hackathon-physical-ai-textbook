# Economy Jetson Kit

Budget-friendly option for edge AI deployment.

## Components

- **NVIDIA Jetson Nano**: $149
- **USB Camera**: $30
- **Basic Sensors**: $50
- **Servo Kit**: $40
- **Power Supply**: $25

**Total**: ~$300

## Capabilities

- Run ROS 2 nodes
- Basic computer vision
- Motor control
- Learning platform

## Limitations

- Lower performance than Orin
- Limited RAM (4GB)
- Slower inference
- Good for learning, not production
